== Description ==
[NeginPardakht](https://NeginPardakht.ir) is one of the Financial Technology 
providers in Iran. NeginPardakht provides multiple payment method and services.
this plugin enables the NeginPardakht's payment gateway for Woocommerce.

Thank you so much for using NeginPardakht Payment Gateway.


== Installation ==
After creating a token on https://neginpardakht.ir and getting an API Key, 
follow this steps:

1. Activate plugin NeginPardakht for Woocommerce.
2. Go tho WooCommerce > Settings > Payments.
3. Enable NeginPardakht payment gateway.
4. Go to Manage.
5. Enter the Token and MerchantId.
